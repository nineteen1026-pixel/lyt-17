## 1. 产品概述

疼痛日记 PWA 是一款面向慢性疼痛患者的个人健康管理应用，帮助用户系统记录疼痛相关数据，分析疼痛规律，为就医和自我管理提供数据支持。

- 核心价值：通过可视化数据分析帮助用户理解疼痛模式，提高生活质量
- 目标用户：慢性疼痛患者、康复期患者、需要跟踪健康状况的人群
- 产品定位：轻量级、离线可用的数据记录与分析工具

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 无需注册，本地存储 | 记录疼痛数据、查看趋势分析、导出数据 |

### 2.2 功能模块
1. **记录主页**：快速记录入口，包含今日概览
2. **部位标注**：人体图点击标注疼痛位置
3. **程度评分**：数字评分量表（0-10分）
4. **诱因记录**：记录可能引发疼痛的因素
5. **用药日志**：记录服用的药物和剂量
6. **运动情况**：记录运动类型和时长
7. **天气关联**：自动记录天气信息
8. **趋势分析**：按周/月生成疼痛趋势图表

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 记录主页 | 今日概览 | 显示今日记录次数、平均疼痛程度、快速记录按钮 |
| 记录主页 | 导航栏 | 底部Tab导航，切换各功能模块 |
| 部位标注 | 人体图示 | 可交互人体图，点击选择疼痛部位，支持多选 |
| 程度评分 | 评分滑块 | 0-10分滑动评分，配合表情图标直观展示 |
| 诱因记录 | 标签选择 | 预设诱因标签（压力、睡眠、饮食等）+ 自定义输入 |
| 用药日志 | 药物表单 | 药物名称、剂量、服用时间、备注 |
| 运动情况 | 运动记录 | 运动类型、时长、强度、感受 |
| 天气关联 | 天气展示 | 自动获取当前天气（温度、湿度、气压） |
| 趋势分析 | 图表展示 | Chart.js 折线图/柱状图，支持周/月切换，多维度对比 |
| 历史记录 | 记录列表 | 按时间顺序展示所有记录，支持筛选和删除 |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：柔和的蓝绿色系 (#3B82F6 主色，#10B981 辅助色)，传达健康、平静的感觉
- **背景**：渐变蓝紫色背景，配合柔和的磨砂玻璃效果
- **按钮风格**：圆角胶囊形按钮，悬停时有轻微上浮和阴影变化
- **字体**：使用 "Noto Sans SC" 作为中文主体字体，清晰现代
- **布局风格**：卡片式布局，内容区有柔和阴影和圆角，营造层次感
- **图标风格**：使用 Lucide Icons，线性简约风格

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 记录主页 | 今日概览 | 大数字展示疼痛评分，环形进度图，卡片式快捷入口 |
| 部位标注 | 人体图示 | SVG人体图，点击高亮，多选标记，浮动标签 |
| 程度评分 | 评分滑块 | 自定义滑块，渐变色彩，表情反馈动画 |
| 趋势分析 | 图表展示 | Chart.js 动画图表，切换按钮，图例交互 |
| 历史记录 | 记录列表 | 时间线布局，可展开详情，滑动删除 |

### 4.3 响应式设计
- **桌面优先**：主内容区最大宽度 1200px，居中布局
- **移动端适配**：断点 768px，单列布局，底部导航优化为触控友好尺寸
- **触控优化**：所有交互元素最小高度 44px，按钮间距充足

### 4.4 动效设计
- 页面切换：淡入淡出 + 轻微位移
- 数据提交：成功反馈动画
- 图表加载：数据渐入动画
- 按钮交互：按下缩放，悬停上浮
